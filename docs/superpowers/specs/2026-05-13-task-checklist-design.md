# 任务打卡系统设计方案

## 项目概述

### 背景
结合育儿知识库，设计一个丰富多样、区分年龄段、难度分级的任务打卡系统，帮助家庭科学育儿、培养孩子好习惯。

### 目标
- 根据孩子年龄段自动推荐适合的任务
- 提供丰富的预设任务库（7大类型）
- 支持进阶式 + 星级难度分级
- 混合模式：智能推荐 + 手动选择

## 功能设计

### 1. 七大任务类型

| 类型 | 图标 | 描述 | 示例任务 |
|------|------|------|----------|
| 规律作息 | 🌙 | 睡眠、饮食、卫生习惯 | 早睡早起、均衡饮食、刷牙洗手 |
| 运动锻炼 | 🏃 | 户外活动、体能训练 | 户外运动、平衡车、球类游戏 |
| 认知学习 | 📚 | 阅读、数学、语言发展 | 亲子阅读、数数游戏、识字启蒙 |
| 社交情感 | 💝 | 情绪管理、社交技能 | 情绪表达、分享合作、礼貌用语 |
| 生活自理 | 🎯 | 独立能力、自我管理 | 自己吃饭、穿脱衣服、整理玩具 |
| 艺术创意 | 🎨 | 绘画、音乐、手工 | 涂鸦绘画、音乐律动、手工制作 |
| 安全意识 | 🛡️ | 自我保护、安全常识 | 交通安全、防拐防骗、急救基础 |

### 2. 年龄分级

#### 3岁任务
- **特点**：基础习惯、简单自理、安全第一
- **难度**：初级为主
- **示例**：
  - 规律作息：每天午睡1-2小时
  - 运动锻炼：户外活动30分钟
  - 认知学习：看图识物
  - 社交情感：认识新朋友
  - 生活自理：自己用勺子吃饭
  - 艺术创意：涂鸦画画
  - 安全意识：不跟陌生人走

#### 4岁任务
- **特点**：能力提升、初步社交
- **难度**：初级+中级
- **示例**：
  - 规律作息：固定时间起床睡觉
  - 运动锻炼：平衡车练习
  - 认知学习：数1-10
  - 社交情感：学会分享玩具
  - 生活自理：自己穿简单衣物
  - 艺术创意：简单折纸
  - 安全意识：记住家庭地址

#### 5岁任务
- **特点**：综合发展、责任培养
- **难度**：中级为主
- **示例**：
  - 规律作息：独立完成睡前准备
  - 运动锻炼：拍球30下
  - 认知学习：认识颜色形状
  - 社交情感：帮助小朋友
  - 生活自理：整理自己的房间
  - 艺术创意：简单绘画创作
  - 安全意识：过马路看红绿灯

#### 6岁任务
- **特点**：入学准备、学习习惯
- **难度**：中级+高级
- **示例**：
  - 规律作息：自己定闹钟起床
  - 运动锻炼：跳绳30下
  - 认知学习：学习握笔写字
  - 社交情感：解决小朋友间冲突
  - 生活自理：整理书包文具
  - 艺术创意：完整绘画作品
  - 安全意识：紧急情况求助

### 3. 难度分级

#### 进阶式分级
- **初级**：基础习惯养成（21天习惯养成）
- **中级**：能力提升（技能进阶）
- **高级**：综合发展（复杂任务）

#### 星级标识
- ⭐ 初级：简单，5分钟内完成
- ⭐⭐ 中级：中等，10-15分钟
- ⭐⭐⭐ 高级：挑战，需要家长配合或较长时间

### 4. 核心功能模块

#### 4.1 智能推荐系统
- **年龄段选择**：首页顶部选择孩子年龄（3-6岁）
- **自动推荐**：根据年龄段推荐适合的任务
- **一键添加**：快速将推荐任务添加到打卡列表
- **推荐理由**：显示任务对应的育儿知识链接

#### 4.2 任务库管理
- **分类浏览**：按7大类型查看所有任务
- **筛选功能**：按年龄、难度、类型筛选
- **搜索功能**：关键词搜索任务
- **收藏功能**：收藏常用任务

#### 4.3 打卡追踪
- **今日进度**：当日完成率
- **本周统计**：本周完成情况
- **本月统计**：本月坚持天数
- **连续打卡**：连续打卡天数

#### 4.4 知识关联
- **任务详情**：显示任务相关育儿知识
- **知识链接**：点击跳转到知识库相关文章
- **科学依据**：说明任务对发展的帮助

## 数据结构设计

### 任务类型枚举
```typescript
type TaskCategory = 
  | 'routine'      // 规律作息
  | 'exercise'     // 运动锻炼
  | 'cognitive'    // 认知学习
  | 'social'       // 社交情感
  | 'selfcare'     // 生活自理
  | 'artistic'     // 艺术创意
  | 'safety';      // 安全意识
```

### 难度等级枚举
```typescript
type DifficultyLevel = 
  | 'easy'         // 初级 ⭐
  | 'medium'        // 中级 ⭐⭐
  | 'hard';        // 高级 ⭐⭐⭐
```

### 任务接口
```typescript
interface Task {
  id: string;
  title: string;
  description?: string;
  category: TaskCategory;
  difficulty: DifficultyLevel;
  ageRange: {
    min: number;  // 最小月龄
    max: number;  // 最大月龄
  };
  duration: number;  // 预计完成时间（分钟）
  knowledgeIds?: string[];  // 关联的知识库文章ID
  frequency: 'daily' | 'weekly' | 'once';
  completedDates: string[];
  isCustom: boolean;  // 是否自定义任务
  createdAt: string;
}
```

## 界面设计

### 任务打卡页面布局

#### 顶部区域
- **孩子年龄选择器**：3岁 / 4岁 / 5岁 / 6岁
- **今日进度卡片**：完成率 + 进度条

#### 主要区域
- **智能推荐区**：根据年龄推荐的任务（可折叠）
- **任务分类区**：7大类型任务卡片列表
- **筛选工具栏**：类型 + 难度 + 搜索

#### 底部区域
- **添加任务按钮**：推荐任务 / 自定义任务

### 交互设计

#### 任务卡片
- 显示：任务名称 + 类型图标 + 星级难度
- 点击：展开任务详情
- 长按/滑动：快速完成打卡

#### 添加任务
- 方式1：点击推荐任务的"添加"按钮
- 方式2：浏览任务库，点击"添加到今日"
- 方式3：点击"自定义任务"创建新任务
